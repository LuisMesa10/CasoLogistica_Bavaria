## Estructura del análisis

El flujo de trabajo se desarrolló en las siguientes etapas:

### 1. Carga de datos
Lectura del archivo Excel desde Google Drive y revisión de la estructura de las hojas.

### 2. Integración de información
Unión entre la tabla de ventas y la tabla maestra de productos mediante la variable `SKU`.

### 3. Limpieza y validación
- Normalización de variables categóricas  
- Conversión de tipos de datos  
- Eliminación de valores nulos  
- Filtrado de los meses relevantes (Mayo y Junio)  

### 4. Análisis exploratorio y agregaciones
- Crecimiento por marca  
- Ventas por tipo de envase  
- Consumo por capacidad  
- Proyección de ventas por SKU  

---

## Resultados principales

A partir del análisis se lograron identificar:

- La marca con mayor crecimiento entre mayo y junio  
- Los envases con mayor y menor participación en ventas  
- La capacidad más consumida en junio
